# Usage

    node heifer.node.js [options] <URL or file ...>

## Options

    -h, --help        output usage information
    -V, --version     output the version number
    -u, --url         Analyze URL (default)
    -j, --json        Parse JSON output from YSlow
    -o, --out [file]  Path to write report to

# Requirements

* phantomjs
* node
